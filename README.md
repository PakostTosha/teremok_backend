# Teremok
Веб-приложение (серверная часть) с интерактивным тренажёром по математике

## Ключевые моменты
- Реализован роутинг для взаимодействия с данными пользователя (CRUD)
- Доступ пользователя к ресурсам реализуется при помощи JWT
